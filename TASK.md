# scx-void 项目开发任务列表

## 1. CLI 层开发 (src/cli/)

### 1.1 项目相关命令 (project.rs)
- **任务**: 实现项目相关的命令，如创建、初始化、添加依赖等
- **验收标准**:
  - 支持 `scx-void project init` 命令用于创建新项目
  - 支持 `scx-void project add <type>` 命令用于添加特定技术栈
  - 使用 clap 库进行命令行参数解析
  - 使用 dialoguer 提供交互式选择界面
- **测试**: 
  - 测试命令行参数解析
  - 测试交互式选择功能
  - 测试命令路由功能

### 1.2 系统相关命令 (system.rs)
- **任务**: 实现系统相关的命令，如关机、重启等
- **验收标准**:
  - 支持 `scx-void system shutdown` 命令用于关机
  - 支持 `scx-void system shutdown --timer=30` 用于定时关机
  - 支持跨平台命令执行
- **测试**:
  - 测试跨平台命令执行
  - 测试命令参数解析

### 1.3 CLI 工具 (utils.rs)
- **任务**: 实现 CLI 相关的辅助功能
- **验收标准**:
  - 提供通用的 CLI 工具函数
  - 包含输入验证、格式化等功能
- **测试**:
  - 测试工具函数功能
